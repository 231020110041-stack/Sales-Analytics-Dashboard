# 📊 Sales Analytics Dashboard

Interactive Sales Dashboard built using Python, MySQL, Pandas, Plotly and Streamlit.

## Features
- KPI Cards
- Sidebar Filters
- Interactive Charts
- Sales Analysis
- Profit Analysis
- Download Reports

## Tech Stack
- Python
- MySQL
- Pandas
- Plotly
- Streamlit

## Installation

```bash
pip install -r requirements.txt
streamlit run script/dashboard/app.py
```
